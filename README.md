# ✈️ Travel Planner (Multi-Agent) – Google ADK & uv

## 📌 Overview
The **Travel Planner (Multi-Agent)** project is an intelligent system that helps users plan trips efficiently using multiple AI agents. Built with **Google ADK (Agent Development Kit)** and **uv**, this project demonstrates how collaborative agents can handle different aspects of travel planning such as itinerary creation, budgeting, and recommendations.

---

## 🚀 Features
- 🤖 Multi-agent architecture for task specialization  
- 🗺️ Smart itinerary generation  
- 💰 Budget estimation and optimization  
- 🍽️ Recommendations for hotels, restaurants, and attractions  
- ⚡ Fast dependency management using **uv**  
- 🧠 Modular and scalable design  

---

## 🏗️ Tech Stack
- Python  
- Google ADK (Agent Development Kit)  
- uv (Python package manager)  
- Large Language Models (LLMs)

## 🧠 How It Works
The system uses multiple agents, each responsible for a specific task:

- **Planner Agent** → Creates travel itinerary  
- **Budget Agent** → Estimates and optimizes cost  
- **Recommendation Agent** → Suggests hotels, food, and attractions  

These agents communicate and collaborate to generate a complete travel plan.
